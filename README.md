# guia_fastapi_vite_postgres

Proyecto FullStack desarrollado con:
- FastAPI
- React + Vite
- PostgreSQL
- Docker

## Ejecutar proyecto

```bash
docker compose up --build
```

## URLs

- Frontend: http://localhost:5173
- Backend: http://localhost:8000
- Swagger: http://localhost:8000/docs

## Tecnologías
- Python 3.12
- FastAPI
- React
- PostgreSQL
- Docker
